# Install Steps For This Story Package

This package adds a new Technology story draft split into body/aside fragments plus metadata and thumbnails.

## Included files
- assets/technology-musk-altman-openai-court-battle-thumbnail.jpg
- assets/technology-musk-altman-openai-court-battle-thumbnail.png
- content/bodies/technology-musk-altman-openai-court-battle.html
- content/asides/technology-musk-altman-openai-court-battle.html
- metadata/master-edition-story-entry.json
- technology-musk-altman-openai-court-battle.md
- technology-musk-altman-openai-court-battle-standalone-preview.html
- AGENT-PROMPT.md

## Quick install
1. Ensure you are on the repository root.
2. Confirm the files above are present.
3. Copy the JSON object from metadata/master-edition-story-entry.json into the stories array in master-edition.json.
4. Run the project build command to regenerate output files.
5. Open technology-musk-altman-openai-court-battle-standalone-preview.html in a local server context and confirm body/aside load correctly.
6. Review image references and replace thumbnails if you have final artwork.

## Notes
- The PNG thumbnail is a valid placeholder file.
- The JPEG thumbnail currently reuses an existing technology image as a temporary asset.
