# The Press preserve-everything upgrade

This package keeps the existing front page, existing article pages, existing assets, and existing layout.

What it adds:
- breaking-news.html
- a breaking rail on the homepage
- stronger click-anywhere story cards
- read-aloud speaker controls on article pages
- AI Newsroom labeling instead of Authors

How to upload:
1. Unzip this package.
2. Open your GitHub repo.
3. Click **Add file → Upload files**.
4. Drag **everything inside the `the-press` folder** into GitHub.
5. Commit changes.
6. Wait about 30 seconds and refresh https://thepress.live

Important: this package is meant to preserve the website. It does **not** remove your existing pages.
